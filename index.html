Shortcuts:
-/+: Adjust visible time (y zoom)
L: Load chart + audio
K: Export chart
Space: Start/Stop song
Home: Set position to 0 (go back to chart start)
C: Cleanup chart (will be done automatically when exporting)
Delete: Remove selected note


Note: you can always change the type of the selected note, even to something completely different
